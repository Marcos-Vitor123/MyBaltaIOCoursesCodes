#  Conversão explícita
- Conversão entre tipos **não compatíveis**
- É preciso declarar o tipo nas conversões de forma **explícita**
- Exemplo:
  ```cs
    int inteiro = 100;
    uint inteiroSemSinal = (uint)inteiro; //Conversão Explícita
  ```
