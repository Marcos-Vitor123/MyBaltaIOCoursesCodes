# Laços de repetição: For
- Significa **Para**.
- Sua finalidade é que **para** cada rodada(interação) nele mesmo, representar um **item**
  - Ele vai funcionar enquanto não atender a sua **condição**
  
```cs
    for (int contador = 0; contador < condicao; contador++)
    {
        //Código aqui
    }
```
