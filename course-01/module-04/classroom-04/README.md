#  Tipos de projeto

- Class Library
    - **DLL** Sem interface
        - `dotnet new classlib`
- Console Aplication
    - Roda no Terminal
        - `dotnet new console`
- Web
    - ASP.NET Web
        - `dotnet new web`
    - ASP.NET MVC
        - `dotnet new mvc`
    - ASP.NET WebAPI
        - `dotnet new webapi`
- Testes
    - Microsoft Tests
        - `dotnet new mstest`
